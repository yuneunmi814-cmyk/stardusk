# 쉼표 · Android 스토어 자산

Google Play 등록용 스크린샷·아이콘. 캡처 환경: Pixel 7 에뮬레이터(1080×2400), 좌표 강릉 경포대.

| 파일 | 화면 | 용도 |
|------|------|------|
| `01_login.png` | 로그인(게스트 + Google로 계속하기) | 휴대전화 스크린샷 |
| `02_explore.png` | 탐색 지도(주변 자연 마커 + 3탭) | 휴대전화 스크린샷 |
| `03_curation.png` | 큐레이션 카드(경포대 · 길찾기 · 안내 듣기) | 휴대전화 스크린샷 |
| `04_saved.png` | 저장한 곳(라이크 모음) | 휴대전화 스크린샷 |
| `05_settings.png` | 설정(로그아웃 · 회원 탈퇴 · 개인정보) | 휴대전화 스크린샷 |
| `play_icon_512.png` | 앱 아이콘 512×512 | Play 등록정보 아이콘 |
| `play_feature_1024x500.png` | 피처 그래픽 1024×500(쉼표 워드마크) | Play 등록정보 그래픽 이미지 |
