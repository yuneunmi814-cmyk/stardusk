# 쉼표 · 안드로이드 실기기/에뮬레이터 테스트 체크리스트

출시 전 동작 검증용. 에뮬레이터(Pixel 7 등) 또는 실기기에서 디버그 APK로 1회 완주합니다.

> 빌드/설치:
> ```bash
> cd android && export JAVA_HOME="/Applications/Android Studio.app/Contents/jbr/Contents/Home"
> ./gradlew :app:assembleDebug
> adb install -r app/build/outputs/apk/debug/app-debug.apk
> ```
> 한국 좌표 주입(에뮬레이터): `adb emu geo fix 128.8961 37.7956` (경포대)

---

## A. 로그인 / 진입
- [ ] 앱 실행 → "쉼 표 / 잠시 멈추어, 숨을 고르다" 로그인 화면 표시
- [ ] **둘러보기(게스트)** · **Google로 계속하기** 두 버튼 모두 보임
- [ ] 게스트로 진입 → 지도 화면 전환
- [ ] (Google) 릴리스 SHA 등록 후 Google 로그인 성공 → 닉네임 반영
- [ ] 위치 권한 팝업 → 허용 시 내 위치로 지도 이동

## B. 탐색(지도)
- [ ] 하단 탭 3개 표시: **탐색 / 저장 / 설정**
- [ ] 지도에 주변 자연 명소 마커 다수 표시(강릉 기준 수십 개)
- [ ] 위치가 한국 밖이면 자동으로 **강릉**으로 폴백(빈 화면 아님)
- [ ] 우상단 내 위치 버튼 동작
- [ ] 좌상단 **위치 칩**(동네명 + 변경) 표시 — 탭 시 위치 설정 화면
- [ ] 위치 설정: 지도 **중앙 핀 이동** 시 동네명 갱신 · **도로명/건물명 검색** 동작
- [ ] **"해당 위치로 시작하기"** → 그 지역 명소로 재검색(같은 좌표 재선택도 재검색)

## C. 큐레이션 카드 ("지금, 가까운 쉼표로")
- [ ] 버튼 탭 → 풀스크린 카드 오버레이 등장("지금, 어디로 도망칠까요?")
- [ ] 카드에 사진·명소명·주소·거리·"숨은 명소" 배지 표시
- [ ] 카드 **오른쪽 스와이프 = 라이크**, 왼쪽 = 패스 → 다음 카드로 진행
- [ ] 카드 하단 **라이크(하트)/패스(X) 원형 버튼**이 보이고 탭으로도 동작
- [ ] **길찾기** 탭 → 오버레이 닫히고 지도에 **도보 경로 폴리라인 + 안내 카드** 표시
- [ ] 안내 카드: 명소명 · 총거리 · 도보 시간 · 회전 안내문(◀ ▶로 단계 넘김) 표시
- [ ] (서버 TMAP 키 미설정 시) "직선 기준" 표기로 폴백 — 빈 화면/크래시 없음
- [ ] 안내 카드 **안내 듣기** → 요약+안내문 TTS · **외부 지도앱** → **길안내 앱 선택** 시트(설치된 네이버/카카오/티맵 + 다른 지도앱)
- [ ] 안내 **종료(X)** → 경로 사라지고 "지금, 가까운 쉼표로" 버튼 복귀
- [ ] **안내 듣기** 탭 → 한국어 TTS 음성 재생
- [ ] **라이크한 카드 직후 바로 닫아도** 저장 목록에 남음 *(스코프 유실 버그 수정 검증)*
- [ ] 덱 소진 시 종료 화면 → 다시 보기/닫기

## D. 저장
- [ ] 라이크한 곳이 **저장 탭**에 썸네일·주소·하트와 함께 표시
- [ ] 저장 없을 때 빈 상태 안내 문구 표시
- [ ] (해제) 저장 항목 해제 시 목록에서 사라짐
- [ ] 항목 **길찾기(화살표)** → 길안내 앱 선택 시트(iOS 저장 탭과 동일)

## E. 설정 / 계정
- [ ] 닉네임 표시(게스트: "둘러보는 여행자")
- [ ] **개인정보 처리방침** 링크 → 브라우저로 정상 오픈
- [ ] 앱 버전 표시(1.0)
- [ ] **로그아웃** → 로그인 화면 복귀
- [ ] **회원 탈퇴** → 확인 후 계정/데이터 삭제 & 로그인 화면 복귀

## F. 세션/네트워크 견고성
- [ ] 토큰 만료(401) 시 자동 재발급 후 요청 재시도(끊김 없음)
- [ ] 백엔드 콜드스타트(첫 요청 지연) 후에도 정상 로드
- [ ] 비행기모드 등 오프라인 시 크래시 없이 빈 상태/재시도

## G. 디바이스/표시
- [ ] 다크모드에서 가독성 유지
- [ ] 다양한 화면 비율(작은 폰/태블릿)에서 레이아웃 깨짐 없음
- [ ] 시스템 뒤로가기 동작 자연스러움
- [ ] 앱 아이콘(쉼표 콤마) 정상 표시

## 해결된 폴리시
> ✅ 큐레이션 카드 하단 라이크/패스 버튼이 하단 내비게이션 바에 가려지던 문제
> → 큐레이션을 전체화면 Dialog로 띄워 내비게이션 바 위를 덮도록 수정.
>
> ✅ 지도 마커를 iOS StarDot과 동일한 초원 점 스타일로 통일(기본 빨간 핀 제거).
