# 쉼표(Comma) · 실기기 테스트 체크리스트 (출시 전)

시뮬레이터에서 검증되지 않는 부분(실제 소셜 로그인·GPS·외부 앱 핸드오프 등)을
**물리 아이폰**에서 1회 확인합니다. 한 항목이라도 막히면 출시 보류.

> 환경: iPhone(iOS 17+) · Xcode 15+ · 번들 ID `app.stardust.ios` · 백엔드 운영(Render)

---

## 0. 준비 (Xcode)
- [ ] iPhone USB 연결 → Xcode에서 기기 선택
- [ ] Target **Signing & Capabilities**: Team 선택, "Automatically manage signing" 체크
- [ ] Capabilities에 **Sign in with Apple** 포함 확인
- [ ] **Release** 구성으로 빌드(운영 API 호스트 확인: `ios/Config/Release.xcconfig`)
- [ ] 기기에서 앱 실행 → 첫 실행 크래시 없음, 홈 화면 아이콘이 **쉼표** + 이름 "쉼표"

## 1. 로그인 / 계정 (Guideline 5.1.1)
- [ ] **게스트 둘러보기**: 로그인 없이 진입 → 탐색/저장/큐레이션 정상
- [ ] **Sign in with Apple**: 실제 Apple ID로 로그인 → 닉네임 표시, 재실행 시 자동 로그인
- [ ] **Google 로그인**: 계정 선택 → 정상 진입
- [ ] **로그아웃**(설정) → 로그인 화면 복귀
- [ ] **회원 탈퇴**(설정 → 회원 탈퇴 → 확인) → 계정/데이터 삭제 + 로그인 화면
- [ ] 탈퇴 후 같은 Apple ID 재로그인 시 신규처럼 정상 동작

## 2. 위치 (When-In-Use / 해외 폴백)
- [ ] 첫 진입 시 위치 권한 팝업 **1회만** 노출, 문구 = "가장 가까운 고요한 자연으로…"
- [ ] **허용** → 현위치 기준 주변 자연 마커 표시(한국 내에서)
- [ ] **거부** → 빈 화면 대신 **강릉**으로 폴백되어 마커 표시
- [ ] (가능하면) VPN/실제 해외 또는 위치 시뮬로 한국 밖 → **강릉 폴백** 확인
- [ ] 설정 → 위치 권한 변경 후 재진입 정상

## 3. 탐색 · 큐레이션 · 저장
- [ ] 지도 **핀치 줌/이동** 정상(실기기 두 손가락)
- [ ] 마커 탭 → 명소 카드(사진/거리/길안내) 표시, **KTO 워터마크 안 보임**
- [ ] "지금, 가까운 쉼표로" → 풀스크린 카드, **라이크/패스 스와이프 + 버튼** 동작
- [ ] 라이크한 곳이 **저장 탭**에 즉시 반영
- [ ] 저장 탭: **하트 탭 삭제** / **편집 다중삭제** / **스와이프 삭제** 모두 동작

## 4. 길찾기 · 안내 듣기
- [ ] **길찾기** → 설치된 지도앱(애플/카카오/티맵) 선택지 → 외부 앱으로 목적지 전달
- [ ] **안내 듣기**(도슨트) → 한국어 음성 재생, 다시 누르면 정지
- [ ] 무음 스위치/볼륨에 따른 재생 확인

## 5. 테마 · UI
- [ ] **라이트/다크 모드** 전환(제어센터) → 양쪽 가독성 OK, 형광 반전 없음
- [ ] 노치/다이내믹 아일랜드/홈 인디케이터 영역 가림 없음
- [ ] 한 손 도달성/탭 영역 적절, 텍스트 잘림 없음

## 6. 신뢰성 (엣지 케이스)
- [ ] **콜드스타트**: 백엔드가 자고 있을 때 첫 진입 → 잠시 후 데이터 표시(또는 재시도)
- [ ] **토큰 만료**: 60분 이상 후 재사용 → 게스트는 자동 복구되어 정상 표시
- [ ] **비행기모드/약한 네트워크** → 크래시 없이 에러/빈상태 처리
- [ ] 백그라운드 → 포그라운드 복귀 시 상태 유지

## 7. App Store 제출 직전
- [ ] Xcode **Archive** → Validate 통과(누락 아이콘/권한/서명 없음)
- [ ] App Store Connect: 표시명 **쉼표**, `docs/APP_STORE_METADATA.md` 복붙
- [ ] **개인정보 설문**(메타데이터 8번 표대로: 이메일·이름·사용자ID·정확한위치·제품상호작용 / 추적 없음)
- [ ] 스크린샷 업로드(`screenshots/`) · 개인정보처리방침 URL 입력
- [ ] (권장) 심사 기간 Render always-on 또는 keep-alive 워크플로 수동 실행
